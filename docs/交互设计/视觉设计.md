# 软件测试平台——整体样式色调设计

---

## 1. 设计目标

为软件测试平台定义统一的视觉色彩体系，确保：

- 管理端与业务端视觉风格一致。
- 不同模式（管理模式、工作空间模式、项目模式）有清晰的视觉区分。
- 状态标识（通过、失败、阻塞、未执行等）色彩语义明确。
- 符合企业级产品的专业、稳重、高效定位。

---

## 2. 色彩体系总览

### 2.1 主色调——专业蓝

品牌主色调采用**专业蓝**，传递可靠、专业、沉稳的产品气质。

| 色阶          | 色值        | 用途                 |
| ----------- | --------- | ------------------ |
| Primary-50  | `#EFF6FF` | 浅色背景、选中态背景         |
| Primary-100 | `#DBEAFE` | 悬停态背景              |
| Primary-200 | `#BFDBFE` | 边框色                |
| Primary-300 | `#93C5FD` | 辅助图标               |
| Primary-400 | `#60A5FA` | 次要按钮               |
| Primary-500 | `#3B82F6` | **主色**：主要按钮、链接、选中态 |
| Primary-600 | `#2563EB` | 按钮悬停态              |
| Primary-700 | `#1D4ED8` | 按钮按下态              |
| Primary-800 | `#1E40AF` | 深色文字背景             |
| Primary-900 | `#1E3A8A` | 深色强调               |

### 2.2 管理端功能色——稳重蓝灰

管理端在品牌蓝基础上，融入**蓝灰色调**，强调全局管理的严肃性和权威性。

| 色阶        | 色值        | 用途    |
| --------- | --------- | ----- |
| Admin-50  | `#F8FAFC` | 页面背景  |
| Admin-100 | `#F1F5F9` | 卡片背景  |
| Admin-200 | `#E2E8F0` | 边框    |
| Admin-700 | `#334155` | 侧边栏背景 |
| Admin-800 | `#1E293B` | 侧边栏文字 |
| Admin-900 | `#0F172A` | 深色标题  |

### 2.3 业务端功能色——活力蓝绿

业务端在主色调基础上，融入**蓝绿色调**，体现日常工作的高效、活力和协作感。

| 色阶            | 色值        | 用途      |
| ------------- | --------- | ------- |
| Workspace-50  | `#F0FDF4` | 空间卡片背景  |
| Workspace-100 | `#DCFCE7` | 空间卡片悬停  |
| Workspace-500 | `#22C55E` | 空间激活标记  |
| Workspace-600 | `#16A34A` | 空间管理员徽章 |
| Project-50    | `#EFF6FF` | 项目卡片背景  |
| Project-500   | `#3B82F6` | 项目激活标记  |
| Project-600   | `#2563EB` | 项目卡片边框  |

## 4. 语义色体系

用于状态标识、操作反馈、结果展示。

### 4.1 状态色

| 语义            | 色值        | 用途           |
| ------------- | --------- | ------------ |
| Success       | `#22C55E` | 通过、成功、已完成    |
| Success-Light | `#F0FDF4` | 通过浅色背景       |
| Danger        | `#EF4444` | 失败、不通过、致命、删除 |
| Danger-Light  | `#FEF2F2` | 失败浅色背景       |
| Warning       | `#F59E0B` | 阻塞、警告、严重     |
| Warning-Light | `#FFFBEB` | 阻塞浅色背景       |
| Info          | `#3B82F6` | 信息、进行中       |
| Info-Light    | `#EFF6FF` | 信息浅色背景       |
| Neutral       | `#94A3B8` | 未执行、待评审、轻微   |
| Neutral-Light | `#F8FAFC` | 未执行浅色背景      |

### 4.2 用例优先级色

| 优先级 | 色值        | 说明       |
| --- | --------- | -------- |
| P0  | `#EF4444` | 红色，最高优先级 |
| P1  | `#F59E0B` | 橙色，高优先级  |
| P2  | `#3B82F6` | 蓝色，中优先级  |
| P3  | `#94A3B8` | 灰色，低优先级  |

### 4.3 缺陷等级色

| 等级  | 色值        | 说明  |
| --- | --------- | --- |
| 致命  | `#EF4444` | 红色  |
| 严重  | `#F97316` | 橙色  |

| 轻微  | `#22C55E` | 绿色  |
