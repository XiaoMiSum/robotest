## 1. 总体布局

### 1.1 业务端整体框架

业务端采用“顶部导航 + 内容区”布局，无左侧侧栏。顶部导航栏分为两行：

- **第一行**：动态菜单区（根据当前模式显示相应功能入口）和右侧图标组。
- **第二行**：当前工作空间名称和面包屑（可选）。

```
┌──────────────────────────────────────────────────────────┐
│ [Logo] [空间信息] [成员管理] [项目列表]                    │
│        [📁 我的空间] [⚙ 空间管理] [🔧 系统管理]           │
│        [🔔 消息] [👤 张三]                               │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                      内容区                               │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 1.2 模式与动态菜单

进入工作空间后，平台处于“工作空间模式”，顶部动态菜单显示：**空间信息**、**成员管理**、**项目列表**。

| 模式     | 动态菜单            | 说明            |
| ------ | --------------- | ------------- |
| 未选中空间  | 引导提示“请选择一个工作空间” | 不显示空间菜单       |
| 工作空间模式 | 空间信息、成员管理、项目列表  | 进入空间后显示       |
| 项目模式   | 功能测试、接口测试       | 进入具体项目后动态菜单切换 |

### 1.3 全局交互规范

- **危险操作**（移除成员、删除项目等）需二次确认弹窗。
- **消息反馈**：Toast提示，顶部居中，3秒消失。
- **表单校验**：前端实时校验，提交时全量校验。
- **加载状态**：列表显示骨架屏，按钮显示loading。

## 2. 我的空间页面

**路由**：`/workspaces`

点击顶部导航栏“📁 我的空间”图标，跳转至该页面。页面展示当前用户归属的所有工作空间，以卡片列表形式展示，支持分页。

### 2.1 页面布局

```
┌──────────────────────────────────────────────────┐
│  我的工作空间                                     │
├──────────────────────────────────────────────────┤
│  ┌──────────────────┐ ┌──────────────────┐       │
│  │ ⭐电商平台测试    │ │   数据平台       │       │
│  │ 负责电商相关测试  │ │ 数据平台测试     │       │
│  │ 角色: 管理员      │ │ 角色: 成员       │       │
│  │ 默认项目: 核心功能│ │ 默认项目: 无     │       │
│  │ 成员:15 项目:5   │ │ 成员:8 项目:3    │       │
│  │ [进入工作空间]   │ │ [进入工作空间]   │       │
│  └──────────────────┘ └──────────────────┘       │
│  ┌──────────────────┐                            │
│  │   支付系统测试    │                            │
│  │   ...            │                            │
│  │   [进入工作空间]  │                            │
│  └──────────────────┘                            │
├──────────────────────────────────────────────────┤
│                < 第1页/共3页 >                    │
└──────────────────────────────────────────────────┘
```

### 2.2 交互说明

| 操作   | 触发方式          | 反馈                                                   |
| ---- | ------------- | ---------------------------------------------------- |
| 翻页   | 点击分页器或键盘方向键   | 卡片列表刷新                                               |
| 进入空间 | 点击[进入工作空间]或卡片 | 设置活跃空间ID → 调用引导接口 → 跳转至默认项目或项目选择页；顶部动态菜单切换为工作空间模式菜单组 |
| 切换空间 | 在卡片上点击进入      | 同上                                                   |

## 3. 工作空间详情页

│  空间信息                                        │

│  ┌─── 基本信息 ──────────────────────────┐       │

│  │  成员数: 15    项目数: 5              │       │
│  │  [保存修改] (仅管理员可见)            │       │
│  └──────────────────────────────────────┘       │
│                                                  │

## 4. 工作空间成员管理

**路由**：`/workspaces/:workspaceId/members`

通过顶部动态菜单“成员管理”进入。

### 4.1 页面布局

```
┌──────────────────────────────────────────────────┐
│  成员管理                                        │
├──────────────────────────────────────────────────┤
│  [👥 成员列表] [🔗 邀请链接]    (管理员可见标签页) │
├──────────────────────────────────────────────────┤
│  ┌─── 成员列表 ──────────────────────────┐       │
│  │  [+ 添加成员] (管理员可见)            │       │
│  │  用户名 │ 角色 │ 加入时间 │ 操作      │       │
│  │ zhangsan│管理员│2026-01-15│[管理员▼][移除]│  │
│  │ lisi    │ 成员 │2026-02-01│[成员▼]  [移除]│  │
│  └──────────────────────────────────────┘       │
└──────────────────────────────────────────────────┘
```

### 4.2 交互说明

**成员列表标签页**：

| 操作   | 触发方式              | 反馈                                       |
| ---- | ----------------- | ---------------------------------------- |
| 添加成员 | 点击[添加成员]          | 弹窗：搜索用户（远程搜索，防抖300ms）+ 多选，提交后显示成功数量和跳过列表 |
| 更新角色 | 下拉切换 member/admin | 降级唯一管理员时阻止并提示“至少保留一个管理员”                 |
| 移除成员 | 点击[移除]            | 二次确认弹窗；移除唯一管理员时阻止                        |
| 自行退出 | 点击自己行的[移除]        | “确定退出该工作空间？”→ 执行后切换空间                    |
| 搜索成员 | 输入框键入             | 前端过滤或后端搜索                                |

**邀请链接标签页**（仅管理员可见）：

| 操作   | 触发方式       | 反馈                                          |
| ---- | ---------- | ------------------------------------------- |
| 创建链接 | 点击[创建邀请链接] | 弹窗：设置过期时间（可选）、最大次数（可选）→ 生成后显示URL + [复制链接]按钮 |
| 复制链接 | 点击[复制链接]   | 一键复制，Toast提示“已复制”                           |
| 撤销链接 | 点击[撤销]     | 二次确认 → 状态变为“已撤销”                            |
| 查看列表 | 切换至标签页     | 显示链接列表：创建者、过期时间、使用次数/最大次数、状态                |

## 5. 项目管理

### 5.1 项目引导页

**路由**：`/workspace/projects/onboarding`  
**触发条件**：切换空间后，该空间无任何项目。

```
┌──────────────────────────────────────────────────┐
│                                                  │
│              🚀 欢迎来到「电商平台测试」           │
│                                                  │
│         当前工作空间还没有任何项目                  │
│         创建您的第一个项目开始测试管理              │
│                                                  │
│            [创建第一个项目]                        │
│                                                  │
└──────────────────────────────────────────────────┘
```

点击按钮弹出项目创建弹窗，创建成功后跳转至项目列表页。

### 5.2 项目选择页

**路由**：`/workspace/projects/select`  
**触发条件**：切换空间后无个人默认项目但空间有项目。

```
┌──────────────────────────────────────────────────┐
│  选择项目                                        │
├──────────────────────────────────────────────────┤
│  ┌──────────────┐ ┌──────────────┐               │
│  │ ⭐ 核心功能   │ │   用户中心    │               │
│  │ 核心模块测试  │ │ 用户模块测试  │               │
│  │ [设为默认]   │ │ [设为默认]   │               │
│  │ [进入项目]   │ │ [进入项目]   │               │
│  └──────────────┘ └──────────────┘               │
└──────────────────────────────────────────────────┘
```

**交互说明**：
| 操作 | 触发方式 | 反馈 |
|------|----------|------|
| 进入项目 | 点击[进入项目] | 设置活跃项目ID → 跳转至项目工作台 |
| 设为默认 | 点击[设为默认] | 调用接口 → 卡片标注⭐，Toast提示 |

### 5.3 项目列表页

**路由**：`/workspace/projects`（通过顶部动态菜单“项目列表”进入）

#### 5.3.1 页面布局

```
┌──────────────────────────────────────────────────┐
│  项目管理                          [+ 创建项目]   │
├──────────────────────────────────────────────────┤
│  [状态: 全部 ▼] [🔍 搜索项目...]                 │
├──────────────────────────────────────────────────┤
│  ┌──────────────────┐ ┌──────────────────┐       │
│  │ ⭐ 核心功能测试   │ │   用户中心改版   │       │
│  │ 活跃 | 2026-08~09│ │ 活跃 | 2026-07~12│       │
│  │ [设为默认][进入] │ │ [设为默认][进入] │       │
│  │ (管理员:归档 删除)│ │ (管理员:归档 删除)│       │
│  └──────────────────┘ └──────────────────┘       │

└──────────────────────────────────────────────────┘
```

#### 5.3.2 交互说明

| 操作    | 触发方式              | 反馈                           |
| ----- | ----------------- | ---------------------------- |
| 搜索/筛选 | 输入关键词或选择状态        | 列表自动刷新                       |
| 创建项目  | 点击[创建项目]          | 弹出表单弹窗（名称、描述、起止时间），校验名称空间内唯一 |

| 归档    | 点击[归档]（管理员）       | 二次确认 → 检查进行中计划 → 执行          |
| 启封    | 点击[启封]（管理员，已归档项目） | 状态恢复为活跃                      |
| 删除    | 点击[删除]（管理员）       | 二次确认 → 检查项目是否为空 → 执行         |

## 6. 邀请加入页面（公开）

**路由**：`/join?token=xxx`

### 6.1 页面布局

```
┌──────────────────────────────────────────────────┐
│                                                  │
│            🔗 加入工作空间                        │
│                                                  │
│        您将被加入「电商平台测试」工作空间           │
│                                                  │
│       邮箱: [________________]                   │
│       密码: [________________]                   │
│       (8-64字符，含大小写、数字、特殊字符)         │
│                                                  │
│              [加入并登录]                         │
│                                                  │
│         已有账号？输入密码验证后直接加入            │
│         没有账号？将自动创建并加入                  │
└──────────────────────────────────────────────────┘
```

### 6.2 交互说明

| 操作   | 触发方式      | 反馈                                                                         |
| ---- | --------- | -------------------------------------------------------------------------- |
| 验证链接 | 页面加载      | 调用验证接口 → 无效则展示错误提示及原因                                                      |
| 提交   | 点击[加入并登录] | 前端校验 → 调用加入接口 → 成功：存储Token，设置活跃空间，`isNewUser=true`时展示欢迎提示，跳转空间内部；失败：显示具体错误 |
| 密码校验 | 实时        | 显示密码强度指示条（弱/中/强）                                                           |
| 防重复  | 提交后       | 按钮置灰显示loading，防止重复提交                                                       |

## 7. 通用交互模式总结

### 7.1 弹窗类型

| 类型    | 使用场景                |
| ----- | ------------------- |
| 确认弹窗  | 移除成员、删除项目、解散空间、撤销链接 |
| 表单弹窗  | 创建/编辑项目、添加成员、创建邀请链接 |
| 选择器弹窗 | 设置默认项目（下拉选择器）       |

### 7.2 加载状态

- **列表加载**：首次骨架屏，刷新顶部进度条。
- **表单提交**：按钮loading+禁用，完成恢复。
- **切换空间**：顶部进度条+主工作区骨架屏。

### 7.3 空状态处理

| 场景    | 展示                 |
| ----- | ------------------ |
| 无工作空间 | “暂无归属的工作空间，请联系管理员” |
| 空间无项目 | 项目引导页（onboarding）  |
| 无默认项目 | 项目选择页（select）      |
| 列表无数据 | “暂无数据”+ 创建按钮       |
| 搜索无结果 | “未找到匹配结果，请调整搜索条件”  |
