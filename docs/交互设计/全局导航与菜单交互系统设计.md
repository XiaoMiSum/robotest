## 1. 概述

本文档定义软件测试平台的全局导航结构、菜单系统及核心交互流程。平台划分为两种顶层模式——**管理模式**和**业务模式**，业务模式进一步细分为**工作空间模式**和**项目模式**，整体采用统一的顶部导航栏 + 动态内容区布局。

---

## 2. 顶部导航栏布局

顶部导航栏固定高度48px，全平台统一，从左至右依次为：

| 位置     | 组件          | 说明             |
| ------ | ----------- | -------------- |
| 最左侧    | 平台Logo      | 固定显示，点击跳转首页    |
| Logo右侧 | 动态菜单区       | 根据当前模式/上下文动态变化 |
| 右侧     | 我的空间（图标+文字） | 始终显示           |
| 右侧     | 我的项目（图标+文字） | 默认隐藏，进入项目后显示   |
| 右侧     | 空间管理（图标+文字） | 默认隐藏，进入空间后显示   |
| 右侧     | 系统管理（图标+文字） | 始终显示，管理模式入口    |
| 右侧     | 消息通知（图标+文字） | 始终显示，未读红点      |
| 右侧     | 用户头像+名称     | 始终显示，下拉菜单含退出   |

```
┌──────────────────────────────────────────────────────────────┐
│ [Logo] [动态菜单区]    [📁 我的空间] [📂 我的项目]             │
│                        [⚙ 空间管理] [🔧 系统管理]              │
│                        [🔔 消息] [👤 张三]                    │
└──────────────────────────────────────────────────────────────┘
```

所有导航栏图标均采用“图标 + 文字描述”的形式，图标使用Element Plus图标库对应图标，文字描述紧邻图标右侧。

## 3. 模式与动态菜单映射

### 3.1 管理模式

**触发**：点击“🔧 系统管理”图标。

| 元素   | 状态变化               |
| ---- | ------------------ |
| 动态菜单 | 切换为：用户管理、角色管理、空间管理 |
| 我的项目 | 隐藏                 |
| 空间管理 | 隐藏                 |
| 系统管理 | 高亮激活态              |
| 主内容区 | 渲染管理模式面板（系统管理功能页面） |

### 3.2 工作空间模式

**触发**：通过“我的空间”页面选中一个空间。

| 元素   | 状态变化               |
| ---- | ------------------ |
| 动态菜单 | 切换为：空间信息、成员管理、项目列表 |
| 空间管理 | 显示                 |
| 我的项目 | 隐藏                 |
| 主内容区 | 渲染工作空间面板（空间管理功能页面） |

### 3.3 项目模式

**触发**：在工作空间面板中通过项目列表进入具体项目。

| 元素   | 状态变化            |
| ---- | --------------- |
| 动态菜单 | 切换为：功能测试、接口测试   |
| 我的项目 | 显示并高亮           |
| 空间管理 | 保持显示            |
| 主内容区 | 渲染项目面板（项目内功能页面） |

### 3.4 未选中空间

| 元素   | 状态变化              |
| ---- | ----------------- |
| 动态菜单 | 显示引导提示“请选择一个工作空间” |
| 我的项目 | 隐藏                |
| 空间管理 | 隐藏                |

## 4. 核心交互流程

### 4.1 进入管理模式

```
点击「🔧 系统管理」图标
  → 动态菜单切换为：用户管理、角色管理、空间管理
  → 隐藏「📂 我的项目」「⚙ 空间管理」图标
  → 高亮「🔧 系统管理」图标
  → 主内容区渲染管理模式面板
```

### 4.2 选择工作空间

```
点击「📁 我的空间」图标
  → 路由跳转至“我的空间”页面（/workspaces）
  → 页面以卡片列表形式展示当前用户所有归属空间
  → 支持分页（page + pageSize），底部显示分页器
  → 点击某空间卡片的“进入”按钮
  → 设置活跃空间ID，动态菜单切换为：空间信息、成员管理、项目列表
  → 显示「⚙ 空间管理」图标
  → 主内容区渲染工作空间面板
```

### 4.3 进入项目

```
在工作空间面板中点击某项目（或通过项目选择页进入）
  → 动态菜单切换为：功能测试、接口测试
  → 显示「📂 我的项目」图标并高亮
  → 保持「⚙ 空间管理」图标显示
  → 主内容区渲染项目面板
```

### 4.4 返回空间

```
点击「⚙ 空间管理」图标 或 动态菜单中的「项目列表」
  → 退出项目面板
  → 隐藏「📂 我的项目」图标
  → 动态菜单恢复为：空间信息、成员管理、项目列表
  → 主内容区恢复工作空间面板
```

### 4.5 从管理模式返回业务模式

```
在管理模式下点击Logo 或 通过「📁 我的空间」选择空间
  → 退出管理模式
  → 恢复对应业务模式的菜单和图标状态
  → 取消「🔧 系统管理」高亮
```

## 5. 页面设计

### 5.1 我的空间页面

**路由**：`/workspaces`

**触发**：点击顶部导航栏“📁 我的空间”图标。

**页面布局**：

```
┌──────────────────────────────────────────────────────────┐
│  我的工作空间                                             │
├──────────────────────────────────────────────────────────┤
│  ┌──────────────────┐ ┌──────────────────┐               │
│  │ ⭐ 电商平台测试   │ │   数据平台       │               │
│  │ 负责电商相关测试  │ │ 数据平台测试     │               │
│  │ 角色: 管理员      │ │ 角色: 成员       │               │
│  │ 默认项目: 核心功能│ │ 默认项目: 无     │               │
│  │ 成员:15 项目:5   │ │ 成员:8 项目:3    │               │
│  │ [进入工作空间]   │ │ [进入工作空间]   │               │
│  └──────────────────┘ └──────────────────┘               │
│  ┌──────────────────┐                                    │
│  │   支付系统测试    │                                    │
│  │   ...            │                                    │
│  │   [进入工作空间]  │                                    │
│  └──────────────────┘                                    │
├──────────────────────────────────────────────────────────┤
│                < 第1页/共3页 >                            │
└──────────────────────────────────────────────────────────┘
```

**分页**：后端接口 `GET /api/workspace/me` 支持 `page` 和 `pageSize` 参数，返回 `{ records: [], total: number }` 格式。前端使用 Element Plus 的 `el-pagination` 组件，默认每页20条。

**卡片交互**：

- 每张卡片显示空间名称、描述、角色标签（管理员/成员）、成员数、项目数、默认项目名称。
- 当前活跃空间卡片有边框高亮。
- 点击“进入工作空间”按钮：设置活跃空间ID → 调用引导接口 → 跳转至默认项目或项目选择页。
- 支持键盘方向键在卡片间切换焦点。

### 5.2 消息通知弹窗

**触发**：点击“🔔 消息”图标。

**内容**：下拉面板展示最近通知列表，点击可查看详情或标记已读。

## 8. 状态管理

全局导航状态通过 Pinia store 统一管理：

- **currentMode**：'admin' | 'workspace' | 'project' | 'none'——当前所处的顶层模式
- **activeWorkspaceId**：当前活跃空间ID
- **activeProjectId**：当前活跃项目ID
- **workspaceList**：用户归属空间列表（分页缓存）
- **dynamicMenuItems**：根据 currentMode 计算属性动态生成菜单项
- **visibleIcons**：根据 currentMode 计算属性控制各图标的显隐和高亮

各图标组件通过响应式状态控制显隐和高亮，动态菜单通过路由或动态组件切换。
